# GitHub Pages Setup Instructions

## Files Created

I've created three HTML files for your GitHub Pages site:
- `index.html` - Home page
- `support.html` - Support page (for Support URL)
- `privacy.html` - Privacy Policy page (for Privacy Policy URL)

## Step-by-Step Setup

### 1. Create GitHub Repository

1. Go to https://github.com
2. Click the "+" icon (top right) → "New repository"
3. Name it: `wirdly-website` (or any name you prefer)
4. Make it **Public** (required for free GitHub Pages)
5. **Don't** initialize with README (we'll upload files)
6. Click "Create repository"

### 2. Upload Files to GitHub

**Option A: Using GitHub Web Interface**

1. In your new repository, click "uploading an existing file"
2. Drag and drop these three files:
   - `index.html`
   - `support.html`
   - `privacy.html`
3. Scroll down and click "Commit changes"

**Option B: Using Git (if you have it installed)**

```bash
cd /Users/jamaal/wirdly
git init
git add index.html support.html privacy.html
git commit -m "Add GitHub Pages files"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/wirdly-website.git
git push -u origin main
```

### 3. Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under "Source", select:
   - Branch: `main`
   - Folder: `/ (root)`
5. Click **Save**

### 4. Get Your URLs

After a few minutes, your site will be live at:
- **Home:** `https://YOUR_USERNAME.github.io/wirdly-website/`
- **Support URL:** `https://YOUR_USERNAME.github.io/wirdly-website/support.html`
- **Privacy Policy URL:** `https://YOUR_USERNAME.github.io/wirdly-website/privacy.html`

Replace `YOUR_USERNAME` with your actual GitHub username.

### 5. Update App Store Connect

When submitting your app, use these URLs:
- **Support URL:** `https://YOUR_USERNAME.github.io/wirdly-website/support.html`
- **Privacy Policy URL:** `https://YOUR_USERNAME.github.io/wirdly-website/privacy.html`

## Customizing

### Update Email Addresses

Edit the HTML files and replace:
- `support@wirdly.app` with your actual support email
- `privacy@wirdly.app` with your actual privacy email

### Update Content

You can edit any of the HTML files directly on GitHub:
1. Click the file
2. Click the pencil icon (Edit)
3. Make changes
4. Commit changes

## Notes

- GitHub Pages is free for public repositories
- Changes may take a few minutes to appear
- Your site will be accessible worldwide
- You can use a custom domain later if desired

## Quick Checklist

- [ ] Uploaded all three HTML files
- [ ] Enabled GitHub Pages
- [ ] Got your URLs
- [ ] Updated email addresses in HTML files
- [ ] Tested URLs in browser
- [ ] Ready to use in App Store Connect!
